- 아쉽게도 북극 상공 오존층 관측 데이터는 누락되어 있음.

# 각 데이터셋 전처리 및 병합
## 빙하 면적 데이터
- 분석 대상 열 선택:
  - 'Year', 'hemisphere', 'Extent'.
- 연도별 그룹화.

## 지구 표면 온도 데이터
- 열 라벨 수정:
  - 'J-D' -> 'Temp'.
- 분석 대상 열 선택:
  - 'Year', 'Temp'.

## 온실가스 농도 데이터
- 열 라벨 수정:
  - 'CFC*' -> 'CFC'.
- 'HFCs*' -> 'HFCs'.
- 분석 대상 열 선택:
  - 'Year', 'CO2, 'CFC', 'HCFCs', 'HFCs'
    - 'CO2': 대기 중 주요 온실가스.
    - 'CFC', 'HCFCs', 'HFCs': 대기 중 주요 오존층 파괴 물질.

## 오존구멍 면적 데이터
- 분석 대상 열 선택:
  - 'Year', 'Hole Area'

## 데이터 병합
- 연도 열 기준으로 병합.
- 북극에 해당하는 행은 오존 관련 데이터 삭제. ('Hole Area')
  - 남극 상공 관측 데이터이기 때문.

# 환경 요인들 간의 상관관계
## 지구 표면 온도와 대기 중 온실가스 농도
- 관련 열 선택 및 결측값 제거:
  - 'Temp', 'CO2'
상관계수:
  - 이미지 첨부
    - 지구 표면 온도와 대기 중 온실가스 농도는 매우 강한 양의 상관관계.
- 산점도 시각화:
  - 이미지 첨부
    - 대기 중 이산화탄소와 같은 주요 온실가스 농도가 짙어질수록 지구 표면 온도 상승.
    - 앞으로도 온실가스 농도는 더욱 짙어지는 추세.

## 오존 구멍 면적과 오존층 파괴 물질 농도
- 관련 열 선택 및 결측값 제거:
  - 'Hole Area', 'CFC', 'HCFCs', 'HFCs'
- 상관계수:
  - 이미지 첨부
- 산점도 시각화:
  - 이미지 첨부
    1. CFC(염화불화탄소):
       - 강한 양의 상관관계.
       - 농도가 짙어질수록 오존 구멍의 면적 증가.
    2. HCFCs(하이드로클로로플루오로카본):
       - 중간정도의 양의 상관관계.
       - CFC에 비해 오존 구멍의 면적 증가량이 적음.
    3. HFCs(하이드로플루오로카본):
       - 약한 양의 상관관계.
       - 오존 구멍에 미치는 영향이 대체로 적음.
    4. 오존 구멍은 예전에 비해 면적이 상승했지만 변동폭이 큼.

# 환경 요인과 빙하 면적 간의 상관관계

  - 'CO2', 'Extent'
- 상관계수:
  - 이미지 첨부
    - 북극 빙하 면적과 대기 중 온실가스 농도는 중간 정도의 음의 상관관계.
- 산점도 시각화:
  - 이미지 첨부
    - 대기 중 이산화탄소와 같은 주요 온실가스 농도가 짙어질수록 북극 빙하 면적 감소.
    - 앞으로도 북극 빙하 면적은 저점을 향해 계속 감소하는 추세.

- 산점도 시각화:
  - 이미지 첨부
    - 다른 요인에 의해 면적이 유지되는 것으로 보임.

# 결론 도출
## 북극 빙하 면적과 환경요인의 상관관계
  - 대기 중 온실가스 농도가 짙어지면서, 지구 표면 온도가 상승함에 따라 북극 빙하 면적이 감소.
## 남극 빙하 면적과 환경요인의 상관관계
- 대기 중 오존층 파괴 물질 농도가 짙어지면서, 오존 구멍 확장으로 인해 지표면에 도달하는 자외선 양이 늘어나 빙하 면적에 영향을 줄 것으로 예상했으나, 분석 결과 상관관계가 없음.
- 성층권의 오존층이 얇아지면서 자외선을 흡수하는 능력이 감소하여 성층권이 냉각되므로, 낮아지는 기온으로 인하여 빙하 면적이 유지되는 것으로 판단.
- 북극 상공의 오존층 관측 데이터는 누락되어 있지만, 남극과 비슷한 결과일 거라 판단.

# 개선할 점 및 강화된 역량
## 개선할 점
### 더 많은 환경 변수 고려
- 해수 온도 변화 데이터
  - 대기보다 더 많은 열을 저장하는 해양이 빙하를 녹이는 영향이 더 클 것으로 예상.
- 태양 복사량 변화 데이터
  - 태양열 반사율이 높은 얼음 면적이 줄어듦에 따라, 태양 복사 흡수량이 더 많아져 빙하 면적 감소를 가속화할 것으로 예상.
### 시각화 개선
- 'Tableau'를 활용해 시각화 퀄리티 향상.

## 강화된 역량
### 데이터 전처리
- 여러 데이터셋을 정리하고 병합하는 과정에서 Pandas 라이브러리를 활용한 데이터 전처리 역량 강화.
### 데이터 분석
- 상관관계 분석을 통해 빙하 면적과 환경 요인 간의 관계를 도출하는 능력 향상.
### 문제 해결
- 예상과 달리 오존 구멍과 남극 빙하 면적 사이의 관계가 미미하다는 점을 인식하고 이를 설명하는 논리적 사고력 향상.
### 시각화 기술
- matplotlib을 사용한 시각화 기술을 통해 데이터를 더 잘 전달할 수 있는 능력 향상.
